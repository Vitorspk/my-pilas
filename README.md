# MyPilas

MyPilas is a simple and practical financial management software for small businesses. This repository contains the landing page for the MyPilas project.

## Features

*   **Homepage:** Displays the main features of the software, a testimonial, and the pricing plans.
*   **Subscription Page:** A form to subscribe to one of the plans.
*   **Responsive Design:** The layout adapts to different screen sizes.

## Technologies Used

*   HTML5
*   CSS3
*   JavaScript

## How to Run

1.  Clone this repository: `git clone https://github.com/vschiavo/my-pilas.git`
2.  Navigate to the project directory: `cd my-pilas`
3.  Open the `index.html` file in your web browser.

## Project Structure

*   `index.html`: The main landing page.
*   `assine.html`: The subscription page.
*   `css/`: Contains the stylesheets.
    *   `base.css`: Basic styles.
    *   `componentes.css`: Component styles.
    *   `layout.css`: Layout styles.
    *   `bootstrap-grid.min.css`: Bootstrap grid system.
*   `javascripts/`: Contains the JavaScript files.
    *   `app.js`: Handles the mobile menu.
*   `imagens/`: Contains the images used in the project.
